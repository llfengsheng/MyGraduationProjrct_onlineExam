# 在线考试系统

#### 项目介绍
该项目主要是针对高中在线考试而开发，前台学生登录考试,后台分为管理员（类似年级主任那种）和普通教师

#### 软件架构
jdk7,8
tomcat7,8
mysql5.1
MyEclipse、Eclipse(idea)
spring
springmvc
mybatis
bootstrap
jquery
jstl


#### 安装教程

1. 下载源码
2. 导入sql文件到数据库，修改数据库账号密码
3. 把源码带入到MyEclipse里面，不会的可以自行百度
4.调整jar文件和配置文件
5.使用Tomcat运行项目即可
